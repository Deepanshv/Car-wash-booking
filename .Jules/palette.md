# Palette's Journal - Car Wash Booking App

## 2025-05-14 - MUI Dialog for Destructive Actions
**Learning:** Using native `window.confirm` breaks the visual immersion of a themed MUI application. Replacing it with a `Dialog` allows for better feedback (like loading states) and consistent styling.
**Action:** Always prefer themed `Dialog` or `Modal` components over native browser alerts/confirms to maintain UX continuity.
