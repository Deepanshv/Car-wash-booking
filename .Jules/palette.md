## 2026-05-17 - Replace native confirm with themed MUI Dialog
**Learning:** Native browser dialogs like `window.confirm` break the visual flow of themed applications and lack support for asynchronous feedback (loading states). Using a themed MUI Dialog improves consistency and allows for a better user experience during destructive actions.
**Action:** Always replace native browser alerts or confirm dialogs with themed components to maintain visual consistency and support advanced features like loading states.
