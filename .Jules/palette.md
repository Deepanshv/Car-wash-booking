# Palette's Journal

## 2025-05-14 - MUI Dialog for Destructive Actions
**Learning:** Native `window.confirm` dialogs break the immersive feel of a themed MUI application and lack support for consistent styling and loading states. Replacing them with a themed `Dialog` allows for better accessibility (ARIA roles) and visual feedback (loading spinners).
**Action:** Use a reusable confirmation dialog pattern or inline MUI Dialogs for destructive actions like deletion to provide a seamless UX.
