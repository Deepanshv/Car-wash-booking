## 2025-05-15 - [Enhancing Booking Form UX and Accessibility]
**Learning:** Providing immediate visual feedback for asynchronous form submissions (via isSubmitting state, CircularProgress in buttons, and Alert for errors) significantly improves the perceived reliability and accessibility of the application. Extracting specific error messages from AxiosError ensures that the user receives actionable feedback.
**Action:** Always implement loading states and error feedback for primary form submissions to improve UX.
