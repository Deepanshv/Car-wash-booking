## 2025-05-15 - [Themed Confirmation Dialogs]
**Learning:** Replacing native browser `window.confirm` with themed MUI `Dialog` components significantly improves UI consistency and provides a more professional feel. Including the entity name (e.g., customer name) in the dialog provides critical context and prevents accidental data loss.
**Action:** Always prefer themed dialogs over native alerts/confirms, and ensure destructive actions have clear context and loading indicators.
