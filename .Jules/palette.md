## 2025-02-16 - Form Submission Feedback
**Learning:** Users need immediate visual feedback when submitting forms to know the action is in progress. Using an 'isSubmitting' state to disable buttons and show a loading spinner, along with explicit error alerts using MUI's Alert component, significantly improves the perceived reliability of the application.
**Action:** Always implement loading states and error handling for asynchronous form submissions using standard MUI components like CircularProgress and Alert.
