## 2026-06-03 - Context-Aware Deletion Dialog
**Learning:** Replacing native browser `window.confirm` with a themed MUI Dialog significantly improves UI consistency and professional feel. Including specific entity details (e.g., customer name) in the confirmation message reduces errors by providing context for destructive actions.
**Action:** Always prefer themed Dialogs over native prompts for destructive actions, and incorporate relevant entity data into the message to prevent accidental data loss.
