## 2025-05-14 - Replace native confirm with MUI Dialog
**Learning:** Replacing native browser `window.confirm` with a themed MUI `Dialog` significantly improves UI consistency and provides a platform for richer feedback, such as including context (e.g., customer name) and showing loading states during asynchronous operations.
**Action:** Use themed Dialog components for destructive actions to ensure a professional aesthetic and prevent users from interacting with the background during critical processes.
